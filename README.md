# Titanic Data Analysis

## Objective
The goal of this project is to analyze passenger data and identify patterns that affect survival, demonstrating how data analysis can support decision-making.

## Tools
- Python (Pandas, Matplotlib)

## Key Findings
- Female passengers had a significantly higher survival rate
- Higher passenger class increased survival chances

## Conclusion
This project demonstrates how data analysis can uncover meaningful patterns and support data-driven decision-making.
